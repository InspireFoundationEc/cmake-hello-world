# cmake-hello-world
### Build
```bash
mkdir build
cd build
cmake ../
cmake --build .
```

### Run
Inside the `build` directory:
```bash
# Windows
./Debug/SquareRootCalc.exe 2

# Linux
./SquareRootCalc 2
```
